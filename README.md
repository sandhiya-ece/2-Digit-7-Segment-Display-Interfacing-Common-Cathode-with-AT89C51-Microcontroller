# 2-Digit 7-Segment Display Interfacing (Common Cathode) with AT89C51 Microcontroller

## Description
This project demonstrates interfacing a 2-digit Common Cathode 7-segment display with the AT89C51 (8051) microcontroller using Embedded C. The microcontroller displays numeric values on the multiplexed 7-segment display. The project is developed using Keil uVision and simulated in Proteus.

## Components Used
- AT89C51 Microcontroller
- 2-Digit Common Cathode 7-Segment Display

## Software Used
- Keil uVision
- Proteus 8
- Embedded C

## Files
- two_digit_7segment.c – Embedded C source code
- two_digit_7segment.uvproj – Keil project file
- STARTUP.A51 – Startup file
- two_digit_7segment.pdsprj – Proteus project file
- circuit.cng - circuit diagram
